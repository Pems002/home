## Sean is here!

简洁风个人介绍页（主页）

[Demo](https://www.imsean.cn)

### At first

因Coding功能升级后全面由腾讯云接管，
且非实名用户无法使用，
故迁移至Github托管，由Vercel提供静态服务

### Features

- 静态Html文件
- 资源文件CDN支持
- Darkmode支持（系统默认值）
- 定制404页面
- 自定义PWA支持
